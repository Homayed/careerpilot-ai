# CareerPilot AI

CareerPilot AI is a full-stack GenAI job application and interview preparation platform.

The project will allow users to upload career documents, paste job descriptions, generate AI-powered job match analysis, create tailored proposals, prepare interview questions, and chat with their own career data.

## Planned Stack

### Backend
- FastAPI
- Async SQLAlchemy
- PostgreSQL
- Alembic
- Celery
- Redis

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS

### GenAI
- OpenAI API
- Embeddings
- pgvector
- Semantic search
- RAG
- Structured outputs
- LangChain
- LangGraph
- Evaluation and guardrails

## Project Goal

The goal of this project is to learn full-stack GenAI engineering step by step while building a portfolio-ready application.
